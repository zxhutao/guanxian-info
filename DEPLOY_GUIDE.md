# 云函数部署指南

## 部署 getUserData 云函数

### 方法一：使用 HBuilderX（推荐）

1. **打开 HBuilderX**
   - 打开项目：`D:\Documents\HBuilderProjects\冠县信息港`

2. **找到云函数**
   - 在左侧目录树展开：`uniCloud-tcb/cloudfunctions/getUserData`

3. **右键部署**
   - 右键点击 `getUserData` 文件夹
   - 选择：`上传并部署` → `云端安装依赖`

4. **等待完成**
   - 底部控制台显示 "上传成功"

5. **验证**
   - 打开微信开发者工具
   - 刷新项目
   - 测试用户中心功能

### 方法二：使用微信开发者工具

1. **打开微信开发者工具**
   - 打开项目：`D:\Documents\HBuilderProjects\冠县信息港`

2. **找到云函数目录**
   - 在左侧目录树找到：`uniCloud-tcb/cloudfunctions/getUserData`

3. **右键部署**
   - 右键点击 `getUserData` 文件夹
   - 选择：`上传并部署：云端安装依赖`

4. **等待完成**
   - 云开发控制台显示部署成功

### 验证部署成功

部署完成后，在小程序中：
1. 登录账号
2. 进入"我的"页面
3. 点击"我的职位"、"我的服务"等
4. 应该能正常显示数据（如果有发布记录的话）
